# Estación Calidad Aire Móvil

Se trata de crear un estación de medida de calidad de aire (basada en la [estación](https://github.com/javacasm/MedidaCalidadAire)) que se puede llevar en una bici/patinete o similar
Inspirado en [Mobisens](https://www.hackster.io/Telina/mobile-fine-dust-pm10-pm2-5-and-no2-meter-5d011b)
Se necesita una caja estanca, salvo por la entrada de aire para los sensores
Hay que evaluar cómo influye la velocidad de movimiento en la medida

Añadimos un GPS para geolocalizar las medidas

Un ventilador y/o panel solar generan electricidad para alimentar la batería
